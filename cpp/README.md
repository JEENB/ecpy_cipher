ecpy Native Module
====================

# What
This is Native module of [ecpy](https://github.com/elliptic-shiho/ecpy/).

# How

```
> make
> sudo cp bin/ecpy_native.so /path/to/install/directory/
```

# Requirements
* g++
 - require C++11 ~ Support
* Boost C++ Library (boost::python)
* Python 2.x Development Package
* GNU MP
